# Sistema de Login

Projeto simples de autenticação desenvolvido com HTML, CSS e JavaScript.

## Funcionalidades
- Tela de login responsiva
- Validação de email e senha
- Feedback de autenticação
- Interface moderna

## Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript

## Objetivo do Projeto
Praticar conceitos básicos de autenticação, manipulação do DOM e desenvolvimento web.

## Como Executar
1. Baixe os arquivos do projeto
2. Abra o arquivo index.html no navegador
